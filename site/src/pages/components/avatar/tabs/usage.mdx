An avatar houses an image that represents the user.

<br />

#### Structure

An avatar consists of the display picture of the entity that it represents i.e. a user.

In case a display picture is not available, it consists of a colored background and the initials of the name that it represents. And if somehow the initials can't be fetched, a fallback glyph is used.
